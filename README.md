# Art Feedback Button

Плагин обратного звонка. Выводит шорткодом нопку обратного звонка. 

Работает на REST API. Кнопка выводиться `[afb]`. 

Дополнительные атрибуты шорткода
* `label` - надпись на кнопке
* `class` - произвольный ccs класс
